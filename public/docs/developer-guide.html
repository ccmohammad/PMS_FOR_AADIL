PharmacyNext AI - Developer Documentation

System Requirements

  • Node.js (v18.0.0 or higher)
  • MongoDB (v4.4 or higher)
  • npm or yarn package manager
  • OpenAI API key (for AI features)

Installation Steps

1. Extract the Download

After purchasing from CodeCanyon, you'll receive a zip file. Extract it to your desired location.

main_files/
├── pharmacy-next.zip/     # Contains the application source code
└── documentation/   # Contains user and developer guides

2. Unzip and Navigate to Source Code

cd pharmacy-next

3. Install Dependencies

npm install

or if you're using yarn:

yarn install

4. Environment Setup

Create a .env.local file in the root directory with the following variables:

MONGODB_URI=your_mongodb_connection_string
NEXTAUTH_SECRET=your_nextauth_secret
NEXTAUTH_URL=http://localhost:3000
NEXT_PUBLIC_OPENAI_API_KEY=your_openai_api_key

5. Database Setup

Ensure MongoDB is running and accessible via the connection string provided in your .env.local file.

6. Build and Start the Application

For development:

npm run dev

For production:

npm run build
npm start

API Documentation

The system provides RESTful APIs for various operations. Here are the main endpoints:

# Authentication
POST   /api/auth/login       # User authentication

# User Management
GET    /api/users            # Get users list
POST   /api/users            # Create new user
PUT    /api/users/:id        # Update user
DELETE /api/users/:id        # Delete user

# AI Features
POST   /api/ai/drug-interactions     # Check drug interactions
POST   /api/ai/product-search        # AI product search
POST   /api/ai/prescription-processing # Process prescription images
GET    /api/ai/inventory-prediction  # Get inventory predictions
GET    /api/ai/expiry-management     # Get expiry alerts
POST   /api/ai/settings              # Update AI settings

# Core Features
GET    /api/inventory        # Get inventory with pagination
GET    /api/products         # Get products with pagination
GET    /api/sales            # Get sales data
POST   /api/sales            # Create new sale
