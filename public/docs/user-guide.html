3. Inventory Management

3.1 Adding Products

To add a new product:

  1. Click "Add Product" button
  2. Fill in product details:
    • Product Name
    • Generic Name
    • Category
    • Manufacturer
    • Unit Price
    • Purchase Price
    • Stock Quantity
    • Reorder Level
    • Expiry Date
    • Batch Number
  3. Upload product image (optional)
  4. Click "Save Product"

Tip: Set up product categories before adding products to maintain better organization.

3.2 Managing Stock

To update stock levels:

  1. Navigate to Inventory > Stock Management
  2. Select product to update
  3. Click "Update Stock"
  4. Enter:
    • Quantity received
    • Purchase price
    • Invoice number
    • Supplier details
    • Batch number
    • Expiry date
  5. Click "Save"

3.3 Stock Alerts

The system provides alerts for:

  • Low stock items (below reorder level)
  • Expiring products (within 3 months)
  • Out of stock items

4. Sales and Billing

4.1 Creating a New Sale

  1. Click "New Sale" button
  2. Search for products:
    • By name
    • By generic name
    • By barcode
  3. Add products to cart:
    • Enter quantity
    • Check available stock
    • View unit price
  4. Apply discounts (if applicable)
  5. Add customer details (optional)
  6. Select payment method
  7. Complete sale

Tip: Use keyboard shortcuts for faster billing:

  • F2: New Sale
  • F3: Product Search
  • F4: Complete Sale

4.2 Managing Returns

To process a return:

  1. Go to Sales > Returns
  2. Enter invoice number
  3. Select items to return
  4. Enter return reason
  5. Process refund

5. AI Features

5.1 AI Prescription Processing

Use AI to automatically process prescription images:

  1. Click "AI Scan Prescription" button in the sales page
  2. Take a photo of the prescription or upload an image
  3. AI will automatically extract:
    • Doctor name and patient information
    • Medication names and dosages
    • Frequency and duration
    • Special instructions
  4. Review the extracted information
  5. Add medications to cart with one click

Tip: Ensure good lighting and clear handwriting for best AI recognition results.

5.2 Drug Interaction Checker

Automatically check for dangerous drug interactions:

  1. Add multiple medications to the cart
  2. Click "Check Interactions" button
  3. AI will analyze all medications for potential interactions
  4. Review any warnings or safety alerts
  5. Follow recommendations for safe dispensing

Important: Always review AI recommendations and consult with a pharmacist for complex cases.

5.3 AI Product Search

Search for products using natural language:

  1. Click "AI Search" button in the sales page
  2. Type your search query in natural language:
    • "headache medicine"
    • "blood pressure medication"
    • "cough syrup for children"
    • "diabetes supplies"
  3. AI will find relevant products based on symptoms and conditions
  4. Review search results with relevance scores
  5. Click to add products to cart

5.4 AI Inventory Predictions

Get AI-powered inventory insights:

  1. Navigate to AI Analytics dashboard
  2. View inventory predictions and demand forecasting
  3. Review reorder recommendations
  4. Check expiry management alerts
  5. Use insights to optimize stock levels

5.5 AI Settings Configuration

Configure AI features to your needs:

  1. Go to AI Settings page
  2. Enter your OpenAI API key
  3. Enable/disable specific AI features
  4. Set interaction severity thresholds
  5. Test AI connection

Note: AI features require a valid OpenAI API key. Contact your administrator for setup assistance.

9. Help and Support

9.1 Common Issues

Printer not working
  1. Check printer connection
  2. Verify printer settings
  3. Restart printer
  4. Contact support if issue persists
Sales not saving
  1. Check internet connection
  2. Verify all required fields
  3. Clear browser cache
  4. Contact support if issue persists
AI features not working
  1. Check if OpenAI API key is configured
  2. Verify internet connection
  3. Test AI connection in settings
  4. Ensure AI features are enabled
  5. Contact support if issue persists
Prescription scanning not accurate
  1. Ensure good lighting
  2. Keep prescription flat and straight
  3. Check handwriting is clear
  4. Try different angles
  5. Contact support if issue persists
